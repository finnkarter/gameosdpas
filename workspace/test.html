게임 실행 방법:

이 게임은 ES6 모듈을 사용하므로 로컬 HTTP 서버가 필요합니다.

방법 1: 간단한 서버 실행

1. 폴더에서 터미널/명령프롬프트 열기

2. 다음 중 하나 실행:

python -m http.server 8000

npx live-server

php -S localhost:8000

방법 2: VS Code Live Server

VS Code에서 Live Server 확장 설치 후 index.html 우클릭 → "Open with Live Server"
